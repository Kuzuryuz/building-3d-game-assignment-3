## คำอธิบาย

โปรแกรมนี้เป็นเกม Dungeon Key Escape ที่พัฒนาด้วย OpenGL โดยผู้เล่นจะต้องควบคุมตัวละครเดินสำรวจภายในเขาวงกตสามมิติ เพื่อค้นหากุญแจและออกไปผ่านประตูทางออก

โปรแกรมประกอบด้วย 2 ด่าน โดยด่านที่ 1 ผู้เล่นต้องค้นหากุญแจ 1 ดอก และด่านที่ 2 ต้องรวบรวมกุญแจให้ครบ 3 ดอก จากนั้นจึงจะสามารถเปิดประตูและผ่านด่านได้ แผนที่ของแต่ละด่านถูกสร้างจาก Tile-based Map ซึ่งประกอบด้วยสัญลักษณ์ต่าง ๆ แทนกำแพง มุม ทางแยก พื้น ตำแหน่งผู้เล่น กุญแจ และประตู

การแสดงผลใช้โมเดล 3 มิติจากไฟล์ .obj ได้แก่ พื้น กำแพงหลายรูปแบบ (แนวนอน แนวตั้ง มุม และทางแยก) ตัวละครผู้เล่น กุญแจที่มีแอนิเมชันลอยและหมุน และประตูแบบเปิด/ปิด ตำแหน่งและการหมุนของวัตถุแต่ละชิ้นถูกคำนวณด้วย 3D Transformation Matrix

กล้องจะติดตามผู้เล่นจากมุมมองด้านบนแบบ Third-person โดยอยู่ด้านหลังและสูงกว่าตัวละครเล็กน้อย การตรวจสอบการชนกำแพงใช้วิธี Sample Point Collision Detection รอบ ๆ ตัวผู้เล่น

### การควบคุม

- **W / ↑** : เดินหน้า
- **S / ↓** : เดินถอยหลัง
- **A / ←** : หันซ้าย
- **D / →** : หันขวา
- **Q** : เดินซ้าย (Strafe Left)
- **E** : เดินขวา (Strafe Right)
- **ESC** : ออกจากโปรแกรม
